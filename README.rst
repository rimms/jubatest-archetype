Jubatest
========

A test tool for distributed enviropment.

How to Use
----------

::

  $ python setup.py sdist install
  $ jubatest -f FILE


Functions
---------

* Remote process control and monitoring
    * start
    * stop
    * is_running
    * ...
* Calculate precision
    * number of train data
    * train time
    * ...
* Benchmark
    * latency
    * throughput
    * ...
* Jubatus monitoring
    * model
    * membership
    * ...
* Reporting
    * assert
    * graph
    * ...
